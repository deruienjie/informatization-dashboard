使用提示
1. 点击「＋ 添加」选择本地文件，或直接把文件拖到上方列表；
2. 编辑后按 ⌘/Ctrl + S 回写；
3. 启用「外部自动重读」时，文件被外部改动会重新加载；
4. 本地路径仅在本机可见，不会被上传。
支持能力： 使用 showOpenFilePicker + FileSystemFileHandle 在已授权浏览器（Chromium / Edge / 桌面 Chrome）中直连本地文件。Safari / Firefox 暂不支持直连，只能用“导入快照”模式编辑后导出。
